# Employee Project Monitoring System

Employee Project Monitoring System is a web-based application designed to manage and monitor employee project assignments, project timelines, task progress, and completion details. The system helps organizations track which employee is assigned to which project, project start and end dates, current status, and overall productivity.

## Features

- Employee management
- Project assignment tracking
- Project start and end date management
- Task progress monitoring
- Project completion status
- Detailed project reports
- Admin dashboard
- Employee-wise project details
- User-friendly and responsive interface

## Project Purpose

The main purpose of this project is to help organizations manage employee projects in a structured way. It allows admins or managers to assign projects to employees, monitor deadlines, track project progress, and view completion details in one place.

## User Roles

### Admin / Manager
- Add and manage employees
- Assign projects to employees
- Set project timelines
- Track project progress
- View project completion reports

### Employee
- View assigned projects
- Update project status
- Add work progress details
- Mark tasks as completed

## Project Workflow

1. Admin adds employee details.
2. Admin creates a new project.
3. Admin assigns the project to an employee.
4. Employee views assigned project details.
5. Employee updates daily/weekly progress.
6. Admin monitors project status and completion timeline.
7. Final project completion report is generated.
